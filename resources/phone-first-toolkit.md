# Phone-First Toolkit

**Planned Summer 2026 · Merrillville pilot — venue confirmation pending**

We are designing this hackathon so a person with **only a phone** can participate and submit a project. Laptops, tablets, and Chromebooks are welcome — they are **not required**.

---

## Bare Bones Toolkit

**Purpose:** For people with low storage, limited phone space, or basic devices.

| Tool | Why it helps |
|------|----------------|
| Web browser | Access Docs, Sheets, Gemini, Replit web |
| Google account | Sign in to Google tools |
| Google Docs (browser) | Project plan, write-up, judging notes |
| Google Sheets (browser) | Data, schedules, budgets |
| Gemini (browser) | Brainstorm, explain, draft pitches |
| Camera app | Demo video and progress photos |
| Notes app | Quick ideas and task lists |
| Email | Team communication and submission links |
| QR scanner (optional) | If not built into your camera app |

**Start here if storage is tight.** Starter activities will work on the browser path.

---

## Fully Loaded Toolkit

**Purpose:** For participants with more phone storage or stronger devices.

Everything in **Bare Bones**, plus optional apps:

| App | Why it helps |
|-----|----------------|
| Google Docs app | Offline-friendly editing |
| Google Sheets app | Spreadsheets on the go |
| Google Slides app | Pitch decks |
| Google Gemini app | AI brainstorming |
| Replit mobile app | Build small web apps |
| Canva app | Visual prototypes |
| Google Drive app | File storage and sharing |
| YouTube app | Upload demo videos |
| GitHub app | Optional for advanced coders |
| Discord / GroupMe / Slack | Only if organizers announce official chat |

---

## Tool quick reference

### Google Docs
Planning, project description, team notes, submission text.

### Google Sheets
Budgets, trackers, schedules, simple data stories.

### Gemini
Brainstorming, explaining concepts, outlining demos, writing pitches. Use responsibly; do not paste passwords or private personal data.

### Replit
Build and share web apps from phone, tablet, or browser.

### Google Drive
Store and link project files for submission.

### Camera / Video
Record a **1–3 minute demo** judges can understand.

---

## Accessibility note

If your phone storage is tight, use the **Bare Bones Toolkit**. We will design starter activities so the browser-based path still works.
