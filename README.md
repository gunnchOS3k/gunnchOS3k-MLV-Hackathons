# gunnchOS3k MLV Hackathons

**Public landing page and knowledge base** for planned **Summer 2026** community hack days in Northwest Indiana — starting with the **Merrillville pilot Hack Day**.

> **You do not need a thousand-dollar laptop to participate.** If you have a phone, curiosity, and willingness to learn, you can join, build, have fun, and compete.

---

## Mission

**gunnchOS3k MLV** is a Northwest Indiana-rooted technology education and innovation organization. We make computing, AI, games, and emerging technology accessible through **beginner-friendly, phone-first** hackathons where students, professionals, and curious community members can learn, collaborate, and build projects they are proud to show.

This repository hosts:

1. A **static website** in [`/docs`](docs/index.html) for GitHub Pages (no build step)  
2. A **markdown knowledge base** in [`/resources`](resources/README.md) for guides volunteers and families can reuse

## Preview locally

No install required:

1. Clone this repository  
2. Open **`docs/index.html`** in your web browser (double-click or File → Open)  

For a simple local server (optional):

```bash
cd docs
python3 -m http.server 8080
# Visit http://localhost:8080
```

---

## Enable GitHub Pages

1. Open **https://github.com/gunnchOS3k/gunnchOS3k-MLV-Hackathons**  
2. Go to **Settings → Pages**  
3. Under **Build and deployment**, set **Source** to **Deploy from a branch**  
4. Select branch **`main`**, folder **`/docs`**  
5. Save — GitHub will publish the site within a few minutes

## How to contribute

We welcome fixes and improvements from community members, students, parents, and volunteers:

1. Fork the repository  
2. Edit files in `docs/` or `resources/`  
3. Keep language **warm, beginner-friendly, and accurate** about pending venue/registration/MLH status  
4. Open a pull request with a short description of your change  

**Do not** commit secrets, personal phone numbers, or private venue contracts.
